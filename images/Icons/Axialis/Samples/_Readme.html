Description

This folder contains sample images created from several image objects. Objects are images which can be freely used in Axialis products. They are used to create attractive icons without drawing skills.

Image Objects are simple graphical elements which are assembled together to create more complex artworks like icons, buttons, sprites and more. Image Objects permit you to be creative without drawing a single pixel. By using your mouse to drag & drop Image Objects and applying effects you can create sophisticated graphics. In the near future all Axialis products will support Image Objects. See this page to get an overview of Image Objects.

We strongly recommend you to use this powerful feature to create your icons. Visit the following topics on our website to read more about Image Objects:

Sample Objects Included in the Folders

The images object included in this folder (and subfolders) are provided for educational purpose. However, if you purchased IconWorkshop, you can use them as any other image object (see license).

In This Folder

In this folder, some sample objects have been created from 256x256 objects that you can find in folder "Large" and subfolders "File Types" and "Overlays". You can use them to create icons directly with different formats. This is the easiest and quickest way to proceed. To do so, follow the procedure below:

1. In the Librarian window (choose View/Librarian if it is not visible), select the folder "Objects/Samples".

2. Right-click on the file "Display Warning.jp2" (for example). A menu opens, choose "New Windows® Icon Project From This Image..." if you want to create a Wndows® icon or choose "New Macintosh® Icon Project From This Image..." if you want to create a MacOS® icon.

3. A large dialog box opens. Select the formats you wish to include in your icon. Click OK.

4. A new icon project document opens with all the formats created automatically. You can retouch the images, add/remove formats...

5. Save your icon, you're done.

In Folder "Extra Large"

The folder "Extra Large" contains 512x512 object samples. These images are intended to be used in Macintosh® OSX 10.5 (Leopard). If you want to create an icon for OSX 10.5, follow the procedure below:

1. Choose File/New/Macintosh® Icon Project. A dialog box opens, select "512x512 (Leopard only)" A new editor window opens with a 512x512 edition area.

2. In the Librarian window (choose View/Librarian if it is not visible), select the folder "Objects/Samples/Extra Large". Using a drag & drop, grab the file "Document.png" and drop it into the 512x512 edition area. The first object is added.

3. You can add a second object (this is the main goal of the objects). Using a drag & drop, grab the file "Document - Text.png" and drop it into the edition area. The resulting image is a hi-res text document icon suitable for OSX Leopard.

4. No let's create the other formats: Right-click on the first and only one format "512x512 RGB/A" in the list of available formats. A menu opens, choose "Add Several Formats From This Image...".

5. A dialog box opens. Select the formats you wish to add in your icon. Click OK.

6. You're done! Save the icon.

In Folder "Large"

The folder "Large" contains 256x256 object samples. These images are intended to be used in Macintosh® OSX 10.5 (Tiger) or Windows Vista™. In 256x256 images, objects are drawn in perspective with all details, light effects and shadows. It is recommended to use these images to create formats 256x256, 128x128, 96x96 and 80x80.

Remark:

You can create smaller formats (48x48, 32x32...) from these images but the details will be lost and the images won't be easily recognizable by the user. For smaller formats, we recommend to use the objects in "Medium" and "Small" folders.

If you want to create an icon with large image formats, follow the procedure below:

1. Choose File/New/Windows® Icon Project (or Macintosh®). A dialog box opens, select 256x256 (Vista). A new editor window opens with a 256x256 edition area.

2. In the Librarian window (choose View/Librarian if it is not visible), select the folder "Objects/Samples/Large". Using a drag & drop, grab the file "Folder - Back.png " and drop it into the 256x256 edition area. The first object is added.

3. Add other objects (this is the main goal of the image objects): Using a drag & drop, grab the file "Folder - Document.png" and drop it into the edition area. Redo the same operation with "Folder - Front.png". You can also add a transparency touch to the front pane of the folder: Without deselecting the last object you've added, choose "Draw/Adjustment Effects/Opacity" or press "O". A dialog box opens, set "-25". Click OK. The image should look like this:

4. Now let's create the other formats: Right-click on the first and only one format "256x256 RGB/A" in the list of available formats. A menu opens, choose Add Several Formats From This Image...

5. A dialog box opens. Select the formats 128x128 and 96x96. Click OK.

6. You're done! Now you can add the other formats from this image, or add the other formats from images in subfolder "Medium" and "Small" (see below).

In Folder "Medium"

The folder "Medium" contains 192x192 object samples. These images are intended to be used in all icon projects. It is recommended to use these images to create formats 64x64, 48x48, 32x32 and 24x24. In 192x192 images, objects are drawn in perspective with medium details, larger borders, limited light effects and shadows.

Remark:

Use image objects in "Large" folder to create formats 80x80 to 256x256. If you want to create a 16x16 format from these images, the details will be lost and the images won't be easily recognizable by the user. For format 16x16, we recommend to use the objects in "Small" folder.

If you want to create an icon with medium image formats, follow the procedure below:

1. Choose File/New/Windows® Icon Project (or Macintosh®). A dialog box opens, select 48x48. A new editor window opens with a 48x48 edition area. You can also open an existing icon project (the one from the previous topic for exemple, see above) and select or create a new format 48x48 (choose Draw/New Image Format or press "Ins" - unselect the option "Creation Based On The Following Format").

2. In the Librarian window (choose View/Librarian if it is not visible), select the folder "Objects/Samples/Medium". Using a drag & drop, grab the file "Folder - Back.png " and drop it into the 48x48 edition area. The first object is added.

3. Add other objects (this is the main goal of the image objects): Using a drag & drop, grab the file "Folder - Document.png" and drop it into the edition area. Redo the same operation with "Folder - Front.png". You can also add a transparency touch to the front pane of the folder: Without deselecting the last object you've added, choose "Draw/Adjustment Effects/Opacity" or press "O". A dialog box opens, set "-25". Click OK. The image should look like this (scale 2:1):

4. Now let's create the other formats: Right-click on the first and only one format "48x48 RGB/A" in the list of available formats. A menu opens, choose Add Several Formats From This Image...

5. A dialog box opens. Select the formats 32x32 and 24x24. Click OK.

6. You're done!

In Folder "Small"

The folder "Small" contains 128x128 object samples. These images are intended to be used in all icon projects. It is recommended to use these images to create only the format 16x16. In 128x128 images, objects are not drawn in perspective without details, very large borders and no light effects.

Remark:

Use image objects in "Medium" and "Large" folders to create formats larger than 16x16. If you create larger formats from these images, the resulting icon won't be satisfying.

If you want to create an icon with 16x16 image formats, follow the procedure below:

1. Choose File/New/Windows® Icon Project (or Macintosh®). A dialog box opens, select 16x16. A new editor window opens with a 16x16 edition area. You can also open an existing icon project (the one from the previous topic for exemple, see above) and select or create a new format 16x16 (choose Draw/New Image Format or press "Ins" - unselect the option "Creation Based On The Following Format").

2. In the Librarian window (choose View/Librarian if it is not visible), select the folder "Objects/Samples/Small". Using a drag & drop, grab the file "Folder - Back.png " and drop it into the 16x16 edition area. The first object is added.

3. Add other objects (this is the main goal of the image objects): Using a drag & drop, grab the file "Folder - Document.png" and drop it into the edition area. Redo the same operation with "Folder - Front.png". The image should look like this (scale 2:1):

4. You're done!
