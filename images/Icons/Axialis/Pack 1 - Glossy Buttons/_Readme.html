Description

Axialis Object Pack #1 is a collection of 150 hi-res objects (read more about objects) which have been designed to create create nice looking glossy icons. By using drag & drop in AxialisIconWorkshop you can easily create icons featuring various shapes, symbols, colors, shadows, inner lights, glossy effects.

The objects are provided in 256x256 PNG format with alpha channel. This high resolution permits to create icons from 256x256 to 16x16 without quality loss. Axialis IconWorkshop permits to create icons containing various sizes for Windows®, Windows Vista™ & MacOS®. The icons can also be saved as optimized GIF, JPEG or PNG images for inclusion in your websites. To read more about the content of the pack, visit this page: http://www.axialis.com/objects/ip_icon_01.shtml

All the object remains properties of Axialis. However, you can use them without royalties to create icons for any purpose as long as you respect the User License Agreement: http://www.axialis.com/objects/agreement.html
